Table Tags

A table is created under < table > < /table > tag.

Item Price

"Food menu" is caption for the table written under < caption > < /caption > tag

There are three main tags under table tag i.e. < tr > , < th > < td > .
tr creates rows of the table, th creates heading of the columns, td is for table data.
Here, "Item" and "Price" is table heading, and others are table data.

Semantics in table tag.

< thead > is used to wrap table header.
< tbody > is used to wrap table body.
< tfoot > is used to wrap table footer.
