Forms in HTML

Forms are used to collect data from user.
Implemented using < form > < /form > tag.

Attributes of form tag

Action attribute

Action attribute is used todefine what action needs to be performed when a form is submitted or where the form data needs to be sent.

< forms action="/action.php" >

Input Element

Used to create multiple form controls.
there are multiple types of input that canbe created using type attribute.

Popular input types are:

  • input type = text (Default)
  • input type = password
  • input type = number
  • input type = time
  • input type = color

Placeholder attribute

Used to tell more information about the input type to the user.

<input type = "text" placeholder = "enter name" >

Label attribute

Label element represents a caption for an item in a user interface.
We form connection between Label and input by using Id attribute in input and for attribute in label.


Button Element

<button> Submit </button>

Type Attributes of button

  • Type: submit; is used to perform submit action in forms.
  • Type: button: is used to perform simple buttom action defined by user.
  • Type: reset: is used to reset the data filled in forms.

Button using input element

Input type = submit

Name attribute

Everything in form is saved as name:value pair.
<input type=text placeholder=Enter-name id=Username name=Username >

Range Input Atttribute

min sets the min value of range
max sets the max value of range
step is value for each level
value sets the default valuue of range.
