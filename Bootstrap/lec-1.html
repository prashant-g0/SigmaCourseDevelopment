What is Bootstrap?

Bootstrap is a popular open-source framework for building responsive, mobile-first web pages. It provides a set of CSS and JavaScript tools that help you create visually appealing and functional websites with ease.

Using Bootstrap

To use Bootstrap, you can include the Bootstrap CSS and JavaScript files in your HTML. You can either download them or link to a CDN. Here's an example of linking to a CDN:

        <link href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">
        <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
        <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

Container (Layout)

A container is a fundamental Bootstrap layout element that helps center and structure your content. It can be fluid or fixed width.

        <div class="container">Your content here</div>
        <div class="container-fluid">Your content here</div>

Button (Component)

Bootstrap provides pre-styled buttons. You can customize their appearance using classes like .btn-primary, .btn-secondary, etc.

        <button class="btn btn-primary">Primary Button</button>

Badge (Component)

Badges are used to highlight important information, often used in conjunction with links, buttons, or other elements.

        <span class="badge badge-primary">New</span>

Alert (Component)

Alerts are used for displaying important messages. They can have different contextual classes like .alert-success, .alert-danger, etc.

        <div class="alert alert-warning">This is a warning alert</div>

Button Group (Component)

A button group is a set of buttons grouped together horizontally or vertically.

        <div class="btn-group">
            <button class="btn btn-secondary">Button 1</button>
            <button class="btn btn-secondary">Button 2</button>
            <button class="btn btn-secondary">Button 3</button>
        </div>

Navbar (Component)

Navbars are used to build responsive navigation menus. They can include links, dropdowns, and other content.

        <nav class="navbar navbar-expand-lg navbar-light bg-light">
            <a class="navbar-brand" href="#">Brand</a>
            <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
                <span class="navbar-toggler-icon"></span>
            </button>
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav">
                    <li class="nav-item active">
                        <a class="nav-link" href="#">Home</a>
                    </li>
                </ul>
            </div>
        </nav>

Card (Component)

Cards are flexible and extensible content containers. They can be used to create various content elements like images, text, and links.

        <div class="card">
            <div class="card-body">
                <h5 class="card-title">Card title</h5>
                <p class="card-text">Some quick example text to build on the card title and make up the bulk of the card's content.</p>
            </div>
        </div>

Grid (Layout)

Bootstrapâ€™s grid system allows you to create complex layouts by using a series of rows and columns. The grid is divided into 12 equal columns.

        <div class="row">
            <div class="col">Column 1</div>
            <div class="col">Column 2</div>
            <div class="col">Column 3</div>
        </div>

Grid (Vertical Alignment)

You can vertically align content within grid columns using utility classes like .align-items-center, .align-items-start, or .align-items-end.

        <div class="row align-items-center">
            <div class="col">Aligned center</div>
        </div>

Form Controls in Bootstrap

Bootstrap provides styling for form controls, including inputs, textareas, and selects, making forms look more appealing.

        <input type="text" class="form-control" placeholder="Text input">

Select in Bootstrap

Bootstrap provides styling for <select> elements, making them look consistent across browsers.

        <select class="form-control">
            <option>Option 1</option>
            <option>Option 2</option>
        </select>

Checkbox and Radio in Form

Checkboxes and radio buttons in Bootstrap are styled to ensure consistency and ease of use in forms.

        <div class="form-check">
            <input class="form-check-input" type="checkbox" id="checkbox">
            <label class="form-check-label" for="checkbox">Checkbox</label>
        </div>
        <div class="form-check">
            <input class="form-check-input" type="radio" id="radio">
            <label class="form-check-label" for="radio">Radio</label>
        </div>

Form Layout

Bootstrap provides several ways to structure forms. You can use grid classes or form-specific classes like .form-row for more control.

        <form>
            <div class="form-group">
                <label for="name">Name</label>
                <input type="text" class="form-control" id="name">
            </div>
        </form>
