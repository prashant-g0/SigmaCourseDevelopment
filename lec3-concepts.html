Inline vs Block Elements

Block:

They take the complete space/width for single box (small or big).
Every new box will be placed in a new line.
Eg: Heading tag, paragraph tag

Inline:

They take only the necessary space or width.
Multiple boxes can be placed in same line if enough space is available.
Eg: Anchor tag, Image tag

Div Element

Content division element.
It is used to group other tag under one tag; also known as container tag.
It is an example of block element

Span Element

This is also a container element but this works as an inline tag.

Hr tag

Horizontal rule.
This tag creates a line when used dividing the page horizontally.

Superscript and Subscript tag

Superscript: adds the characters in form of power.
Useful while writing mathematical equations.
Eg: a^2 + b^2 = c^2 | a 2 + b 2 = c2

Subscript: adds the character in lower power.
useful while writing chemical equations.
Eg: H2O | H 2 O

Semantic Markup

Semantic tags

It is markup that relates to meaning of content.
The tags by whhich we can understand the content inside it.
Eg: heading tag, image tag, paragraph tag.

Non-Semantic tags

The tags by which we doesn't directly understand the content inside but does the same work.
Eg: div tag, span tag.

HTML entities

Keywords can never be directly used in html, to print them we use html entities.
Eg: less than symbol < or Multiple       spaces or ampersand & or quotes "hi!"
