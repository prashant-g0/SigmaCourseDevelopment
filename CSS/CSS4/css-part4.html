Alpha channel

There is one more channel in rgb color style and that is alpha channel rgba.
The alpha channel is used for styling the opacity of the color.

The value of alpha is in range of 0-1.
The range of rgb is from 0-255.
rgba(100, 200, 250, 0.3)

Opacity

This stylinng property is used to set the visibility of the element just like how alpha sets the opacity of color.

The value of opacity is in range of 0-1.
h1 { opacity: 0.1; }

CSS Transitions

Transition enables you to define the transition between two states of an element.
h1 { transition: 1s; }
propertyName | duration | timingFunc | delay
eg: element1 { transition: margin-top 2s ease-in-out 0.2s; }

CSS Transform

This property lets you 'rotate', 'skew', 'scale' or 'translate' an element.

1. rotate: element { transform: rotate(45deg);}
2. scale: element { transform: scale(0.2, 0.2);}
3. translate: element { transform: translate(50px, 50px);}
4. skew: element { transform: skew(45deg);}

Box shadow

It adds shadow effects around an elements frame.
box-shadow: 2px 2px 10px green
box-shadow: x-offset y-offset blurness shadow-color

Background image

Let's you add image as a background for div/section.
background-image: url("image.jpg");
along with background image another property is used called,
background-size: contain/cover/auto;

Position property

Sets how an element is positioned in the document.
The top, right, bottom, left, property determine the final location of positioned elements.

  • static: the default value of position is static.
  • reltive: the offset is relative to itself based on the value of top, right, bottom, left.
  • absolute: it is positioned relative to its closest positioned ancester if ny,otherwise relative to the initial containing block.
    no space is given for element positioned as absolute.
  • fixed: it is positioned relative to the initail containing blockestablished by the viewport.
    no space is given for element positioned as fixed.
