Box Model

  • Height: by default it sets the area height of the element.
  • Width: by default it sets the coontent area width of the element.
  • Border: used to set border to elements.
    • border-width
    • border-style
    • border-color
    • border: width style color;
    • border-radius: used to round the corners.
  • Padding: sets the space between content and box
  • Margin: sets the space between two individual box.

Display

It sets whether an element is treated as block or inline element and the layout used for its children.

display: inline;
display: block;

Inline-Block

display: inline-block
This property is used to inline the block elements without disturbing block styling.
