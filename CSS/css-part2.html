Universal selector - *

Used to select everything a document.
* {
property: value;
}

Element selector

Selects all the elements of same type.
tagName { property: value; }

ID selector

Selects an element based on the value of the element's ID attribute.
#id { property: value; }
The id name is case sensitive.

Class selector

Select an element based on their class attribute.
dot(.)className { property: value; }

Descendant selector

select the tag inside another tag.
for eg: div p { property: value; },
this style will be applied to p-tag and not to div-tag.

Adjacent sibling combinator

Selects the element2 which are immediatly after element1.
p+h3 { property: value; },
this selects all the h3-tag appearing after p-tag.

Child combinator

selects all the element2 which are direct children of element1.
span > button { property: value; },
this selects all and only the button-tag 1 level descendent to span-tag.

Attribute selector

Selects elements based on the presence or value of a given attribute.
input [attr="value"] { property: value; }

Psuedo class

A keyword added to a selector that specifies a special state of the selected elements.

  • :hover :: changes style on hover
  • :active :: changes style on click
  • :checked :: changes the style of radio buttons and checkbox.
  • :nth-of-type :: changes the style of nth selected element

Psuedo Elements

A keyword added to a selector that tells you style a specific part of the selected elements.

  • ::first-letter - Styles the first letter of the selected element.
  • ::first-line - Styles the first line of width of the selected element.
  • ::selection - Styles the selection indicator when selecting things on page.

Specificity

id > class > element

Cascading

An algorithm of CSS which decided which property to implement on the element based on the sequence they are defined.
The last defined property for the same element is implemented.
eg. if we style h2 color as black and later we again define h2 color blue then blue is the final color.

!important

To show the most specific things in document.
h2 { bg-color: blue !important; }
If any style is followed by !important keyword then we can never change it's style.
