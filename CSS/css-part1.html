CSS: Cascading Style Sheet

CSS describes the Style of the content i.e. HTML.
HTML acts as a skeleton and CSS makes the skeleton look good and appeling.

Basic format of CSS

h1{
color: red;
}

Here, h1 is selector (selectes which tag or id is accountable for css)
color is property (this will be applied to the selector content)
red is value(for the property)

How to include CSS styles

1. Inline css: writing style directly inline on each element in html file.
< h2 styles="color:blue;"> Content </h2>

Content

2. Using style tag: style is added using < style > tag in html file inside head tag.
< style >
h3{
color: pink;
}
< /style >

Content

3. External stylesheet: writing style in a separate css file and linking with html file.
<head>
<Link rel="stylesheet" href="style.css" >
</head>

Color systems

Named color eg. color:blue;
Each browser recognizes at least 140 named colors.

RGB color eg. rgb(140,220,150
Red Green Blue : 0-225

Hexadecimal or Hex codes eg. color:#ffffff;
Every two digits indicate RGB.

Text Properties

1. Text align: Used to ensure the alignment of text (left, right, center, justify).

2. Text weight: Used to ensure the boldness and lightness of a text. (100-900: normal = 400, bold = 700, 100, bolder, lighter)

3. Text decoration: Used to design the text (underline, strike, overline, etc)
We can creatively play with text decoration by changing colors and styles.
text-decoration-color: red;
text-decoration-styles: wavy;

4. Line height: Ensures the spacing between two lines

5. Letter spacing: Ensures the spacing between letters.

6. Font size: Defines the size of text.

Units in CSS

1. Absolute: px, pt, pc, cm, mm, in
1 inch = 96px
1 inch = 72pt
12 pt = 1pc
Not suitable for responsiveness.

2. Relative: %, em, rem, ch, vh, vw, and many more..
Relative to parent.
