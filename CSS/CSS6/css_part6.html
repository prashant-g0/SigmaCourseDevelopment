CSS Grid

Setting a container's display to grid will make all children grid items.
container {
diaplay: grid;
}
Grid and Flexbox properties are only applied to direct child nd not on indirect child.

Grid Template

They define the lines and track sizing.

  • grid-template-rows: Used to define the number of rows for grid model. The size for each row is given individually.
  • grid-template-columns: Used to define the number of columns for grid model. The size for each column is given individually.

    • 'auto' value can also be used for the properties but not preferred for best practice.

Grid Template: repeat property

Repeat is used to divide all available space.

  • grid-template-rows: repeat(count, 1fr);
  • grid-template-columns: repeat(counnt, 1fr);
  • eg. grid-template-rows: repeat(2, 1fr);
  • eg. grid-template-rows: 1fr, 1fr;
    • fr meaning 1 fraction of 100% available space. width in terms of rows and height in terms of columns.

Grid gaps

This is used to define space b/w grid cells.

  • row-gap: 10px;
  • column-gap: 10px;
  • grid-gap: rowGap, colGap;

Grid columns

Defines an item's starting and ending position inside the column.

  • grid-column-start: line_number;
  • grid-column-end: line_number;
  • grid-column: start_col/end_col;
  • grid-column: start_col/span number;
  • eg. grid-column: 1/4; here we have described the line number.
  • eg. grid-column: 1/span 3; here we have described the total number of box

Grid rows

Defines an item's starting and ending posintion inside a row.

  • grid-row-start: line_number;
  • grid-row-end: line_number;
  • grid-row: start_row/end_row;
  • grid-row: start_row/span number;
  • eg. grid-row: 1/4; here we have described the line number.
  • eg. grid-row: 1/span 3; here we have described the total number of box

Grid properties

  • jsutify-items: used in container to give horizontal alignment.
  • justify-self: used for specific items to align horizontaly.
  • align-items: used in container to give vertically alignment.
  • align-self: used in specific items to align vertically.
  • place-items: used in container as a combined property of justify-items and align-items when the value for both is same.
  • place-self: used in container as a combined property of justify-self and align-self when the value for both is same.

Media queries

Helps create a responsive website compatible with all devices and screen sizes.

@media (min-width: 400px) {div{height:20px;}}
This will cause change in height of div element when screen width is min 400px.

z-index

It decides the stack level of the elements.
Overlapping elements with a larger z-index covers those with smaller ones.

  • z-index: auto(0);
  • z--index: 1/2;
  • z--index: -1/-2;
