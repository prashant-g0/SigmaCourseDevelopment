Flexbox

It is one dimensional layout method for arranging items in a row or column.
element {display: flex;}
The elements inside the parent element are now restricted to fit into the parent element block and does not overflow.
The default direction is "row" and we can always change the direction to column using "flex-direction" property.

The other direction values are: row (left-right), row-reverse (right-left), column (top-bottom) and colummn reverse (bottom-top). The flex properties are given to the parent element.

Justify Content

Tells us how the browser distributes space between and around content items along the main axis.

  • justify-content: flex-start;
  • justify-content: flex-end;
  • justify-content: center;
  • justify-content: space-between;
  • justify-content: space-around;
  • justify-content: space-evenly;

Flex wrap

Sets whether flex items are forced onto one line or can wrap onto multiples lines.

  • flex-wrap: nowrap; (default)
  • flex-wrap: wrap; (along main axis)
  • flex-wrap: wrap-reverse; (along main axis but reversed)

Align items

Distributes our items along cross axis.

  • align-items: flex-start; (to the top of container)
  • align-items: flex-end; (to the bottom of container)
  • align-items: center; (to the center of container)
  • align-items: baseline; (aligns the items keeping the text line at same height)

Align Content

It sets the distribution of space betweena and space around content items along a flexbox's cross bar.

  • align-items: flex-start;
  • align-items: flex-end;
  • align-items: center;
  • align-items: space-around;
  • align-items: space-evenly;
  • align-items: baseline;

Align self

Aligns an item along cross axis. This property is for individual item/elements.

  • align-self: flex-start;
  • align-self: flex-end;
  • align-self: center;
  • align-self: baseline;

Flex sizing

  • flex-basis: 100px; (it sets the initial main size of a flex item.)
  • flex-grow: 1; (it specifies how much of a flex container's remaining space should be assigned to the flex item's main size.)
  • flex-shrink: 1; (it specifies the shrinking of flex item when overflow is encountered.)

Flex shorthand

  • flex grow | shrink | basis: {flex: 2 2 100px;}
  • flex grow | basis: {flex: 2 100px;}
  • flex grow: {flex: 2;} (unitless)
  • flex basis: {flex: 100px;}
